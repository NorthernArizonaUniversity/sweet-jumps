Configuration

Template variables should be explicitly declared in each route, but for convenience, the config / app settings can be accessed using either the config global (all configuration), or the settings global (just app settings.)

Using config

    {% for key, value in config.app %}
  • {{key}}: {{value}}
    • {% endfor %}

Using settings

    {% for key, value in settings %}
  • {{key}}: {{value}}
